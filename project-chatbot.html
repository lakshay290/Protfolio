Random Fact Chatbot

Fun and interactive web-based chatbot with neon aesthetics

Overview

A delightful web-based chatbot featuring a vibrant cyberpunk aesthetic with neon-glowing UI elements. Users can request random fascinating facts from various categories including Animals, Science, History, Space, Food, and Gaming. Perfect for learning interesting trivia or just having fun!

Key Features

  • Category Support: Browse facts from 6+ categories (Animals, Science, History, Space, Food, Gaming)
  • Neon UI Design: Glowing cyberpunk aesthetic with smooth animations and transitions
  • Interactive Interface: Simple and intuitive text-based interaction
  • Real-Time Updates: Facts load instantly with visual feedback
  • Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
  • Pure Frontend: No backend required - fast and lightweight
  • Rich Animations: Smooth transitions and delightful micro-interactions
  • Educational Value: Learn interesting facts while enjoying a beautiful interface

User Experience

Getting Started: Simply choose a category and click "Get Fact" to receive a random fact from that category.

Visual Design: The interface features neon glows, smooth transitions, and a modern cyberpunk aesthetic that makes learning fun.

Mobile Friendly: Fully responsive design that adapts to any screen size.
