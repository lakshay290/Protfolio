Event Reminder System

C++ console application for event management and reminders

Overview

A command-line event management system built in C++ that allows users to add, view, delete, and receive reminders for events. The application features persistent file-based storage ensuring data is retained between sessions, and a colorful ANSI-based console UI for enhanced user experience.

Key Features

  • Add Events: Create new events with title, date, time, and description
  • View Events: Display all saved events in an organized, formatted list
  • Delete Events: Remove events that are no longer needed from the database
  • Event Reminders: Get notifications and alerts for today's scheduled events
  • Date Matching: Automatic matching of events to today's date for reminders
  • Persistent Storage: File-based storage system that preserves events between sessions
  • Colorful Console UI: ANSI-based color codes for improved readability and aesthetics
  • User-Friendly: Simple menu-driven command interface with clear options
  • Data Backup: All event data safely stored in local files with backup capabilities

User Interface

Main Menu: Options to add new events, view existing events, delete events, check today's reminders, and exit the application

Color Scheme: Different colors for different event types and statuses - green for upcoming events, yellow for today's events, red for overdue events

Display Format: Events shown in a table format with columns for date, time, title, and status

Technologies Used

C++

Core programming language for system-level performance

File I/O

Reading and writing events to persistent storage files

STL

Standard Template Library for vectors, strings, and data structures

Console API

Windows/Linux console functions for color output

ANSI Colors

ANSI escape codes for cross-platform color support

Date/Time

C++ chrono library for date and time handling

How It Works

Event Creation: User enters event details through the console menu interface

Storage: Events are saved to a file in JSON or CSV format for persistence

Retrieval: Application loads saved events when started

Reminders: System checks today's date against event dates and displays reminders

Management: Users can modify or delete events as needed

Use Cases

  • Personal task and event management
  • Appointment scheduling and tracking
  • Important date reminders (birthdays, anniversaries)
  • Project deadline tracking
  • Meeting and event organization
