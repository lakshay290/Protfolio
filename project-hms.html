Hospital Management System

A scalable backend system for managing complex healthcare workflows

Overview

A comprehensive backend system for managing complex healthcare workflows. This Hospital Management System (HMS) is built with Django and Django REST Framework, providing a robust API for managing appointments, prescriptions, billing, and role-based access control across different user types (Admin, Doctor, and Patient).

Key Features

  • Appointment Management: Schedule and track patient appointments with doctor assignments and automated reminders
  • Prescription System: Digital prescription generation, tracking, and management for patients
  • Billing Module: Automated billing and payment tracking for procedures, treatments, and services
  • Role-Based Access Control: Separate dashboards and permissions for Admin, Doctor, and Patient roles
  • Security: JWT authentication for secure API endpoints with token-based authorization
  • Performance: Redis caching layer for fast data retrieval and reduced database load
  • Database: PostgreSQL for reliable data storage and complex relational queries
  • Deployment: Docker containerization for consistent environments across different systems
  • Cloud Hosting: Deployed on Railway for automatic scaling and high availability

Technologies Used

Backend

Django & Django REST Framework (DRF) for building robust and scalable APIs

Database

PostgreSQL for complex data relationships and ACID compliance

Caching

Redis for in-memory caching and session management

Containerization

Docker for containerization and consistent deployment

Server

Gunicorn as WSGI server for production deployment

Authentication

JWT (JSON Web Tokens) for secure API authentication

Architecture Highlights

RESTful API Design: Clean and well-documented endpoints following REST principles for easy integration.

Scalability: Designed to handle increasing patient load with caching strategies and database optimization.

Security: JWT tokens, role-based access control, and secure data handling ensure patient privacy.

Cloud-Ready: Docker containerization makes it easy to deploy on any cloud platform or on-premises.
